0. RESET FILE lock.txt IF CONTAINS SOMETHING<BR/>
1. Choose a folder where put image<BR/>
2. Connect your telegram bot then get id of channel<BR/>
3. Choose period (in second)<BR/>
4. Launch script named launcher.py<BR/>
5. That's all...
